📝 Blogging Platform (Java OOPS)

This is a simple blogging platform developed using Core Java and Object-Oriented Programming concepts.

🚀 Features

👤 User Management

User Registration
Login / Logout
View Profile
Edit Profile

📝 Post Management

Create Post
View Posts
Timestamp for each post

📰 Feed Management

Displays all posts
Sorted by latest timestamp

❤️ Like Management
Like / Unlike posts
Displays total like count

💬 Comment System

Add comments
Delete own comments
View comments

🔔 Notifications

Notification when someone likes your post
Notification when someone comments

🧠 OOP Concepts Used

Encapsulation → Private variables with getters
Abstraction → Interfaces (Likeable, Commentable)
Inheritance → Extendable class structure
Polymorphism → Interface method implementation

🛠️ Technologies Used

Java
OOP Concepts
Collections Framework (List, Set, ArrayList, HashSet)

Main.java
User.java

Feed.java
Notification.java
Likeable.java
